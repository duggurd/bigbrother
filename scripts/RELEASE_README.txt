====================================================
  BigBrother - Window Focus & Activity Tracker
====================================================

Thank you for downloading BigBrother!

This package contains two applications:
  1. BigBro Viewer    - GUI application to view your sessions
  2. BigBrother Monitor - CLI application for tracking (optional)

====================================================
  QUICK START
====================================================

Option A: INSTALL TO START MENU (Recommended)
----------------------------------------------
1. Right-click 'install.ps1'
2. Select 'Run with PowerShell'
3. Press any key when prompted
4. Search 'BigBro' in Windows Start Menu

Option B: PORTABLE USE
----------------------
Just double-click 'bigbro.exe' to run!
No installation needed.

====================================================
  FEATURES
====================================================

* Track window focus and active applications
* Record window title changes
* Session-based organization
* Filter unwanted programs
* Add comments to sessions
* Delete sessions you don't want to keep
* Beautiful timeline view

====================================================
  HOW IT WORKS
====================================================

1. Click "Start Session" in BigBro Viewer
2. Add a comment describing what you're working on
3. Work normally - your window focus is tracked automatically
4. Click "Stop Session" when done
5. View your activity timeline!

Your data is stored at:
  %APPDATA%\BigBrother\focus_log.json

====================================================
  USAGE TIPS
====================================================

* Right-click a session to delete it
* Right-click a program to add it to filters
* Use the Settings button to manage program filters
* Sessions are grouped by date for easy navigation
* Icons show which applications you used

====================================================
  UNINSTALLATION
====================================================

To remove Start Menu shortcuts:
  - Run 'uninstall.ps1'

To completely remove BigBrother:
  1. Run 'uninstall.ps1' (removes shortcuts)
  2. Delete this folder
  3. (Optional) Delete: %APPDATA%\BigBrother
